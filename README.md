## Marketing Performance & Customer Conversion Analysis

An end-to-end marketing analytics project focused on analyzing customer behavior, campaign performance, conversion trends, and customer engagement using SQL, Python, Power BI, and DAX.

## Project Overview

This project analyzes 50K+ customer interactions to identify customer behavior patterns, seasonal trends, and conversion opportunities.

The analysis combines exploratory data analysis, customer segmentation, SQL-based data analysis, and interactive Power BI dashboards to generate actionable marketing insights.

## Key Features

- Analyzed 50K+ customer interactions using SQL and Python
- Performed Exploratory Data Analysis (EDA) and customer segmentation
- Identified customer behavior and seasonal conversion trends
- Built an interactive Power BI dashboard with 15+ KPIs
- Developed 10+ DAX measures for advanced business analysis
- Tracked conversion rate, CTR, campaign performance, and customer engagement
- Improved conversion rate from 5.0% to 10.2%
- Maintained a 15.37% Click-Through Rate (CTR)
- Improved customer review scores from 3.7 to 4.0

## Business Impact

The analysis identified key customer behavior and conversion patterns, contributing to an increase in conversion rate from **5.0% to 10.2%** while maintaining a **15.37% CTR**.

Data-driven insights also helped improve customer review scores from **3.7 to 4.0**.
